Despre Drivere Actuatori Auto

Un actuator (dispozitiv de acționare) este o componentă a unei
mașini care este responsabilă pentru deplasarea sau controlul unui
mecanism sau sistem,
de exemplu prin acționarea (deschiderea sau închiderea) a unei supape;
în termeni simpli, este un "mișcător".
Un actuator necesită un semnal de comandă și o sursă de energie.
Semnalul de control este o energie relativ scăzută și poate fi tensiune
sau curent electric, presiune pneumatică sau hidraulică sau chiar
energie umană.
Sursa principală de energie furnizată poate fi curentul electric,
presiunea hidraulică sau presiunea pneumatică.
Când semnalul de comandă este recepționat, servomotorul răspunde prin
transformarea energiei în mișcare mecanică.
Un actuator este mecanismul prin care un sistem de control acționează
asupra unui mediu.
Sistemul de control poate fi simplu (un sistem mecanic sau electronic
fix), bazat pe software (printer driver, robot control system), un om
sau orice altă intrare
